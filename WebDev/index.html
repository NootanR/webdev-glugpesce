What is GLUG?

GLUG” stands for GNU/Linux Users Group.  As the name says, it is a student  chapter who dedicatedly use linux and linux flavored operating systems.


  

Motto of GLUG-PESCE

As it is an organization under FSMK, its main motto is that of FSMK i.e., “to spread awareness about freedom software and its ideals”. Now, what is free software? It is a computer software that gives users the freedom to run the software for any purpose as well as to study, change, and distribute the software and the adapted versions. It is important now than ever before because using freedom software means having control over your own life. It is like having power. On the other hand we have proprietary software that steals the power of knowledge from users. An unacceptable danger to a free society.
So, GLUG-PESCE has joined hands with FSMK and is striving to promote freedom software ideology. It is our duty to wake the people out there who still remain ignorant about the importance of free software and make them adopt it, so that we can achieve a “free society” completely outridden of proprietary software.
